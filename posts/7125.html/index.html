еүҚиЁҖ

дёәд»Җд№ҲиҰҒжҠҳи…ҫиҝҷдёӘпјҹ

дҪңдёәдёҖдёӘеҚҡе®ўдҪңиҖ…пјҢеӣҫеәҠзҡ„йҖүжӢ©дёҖзӣҙжҳҜй—®йўҳпјҡ

  • GitHub + jsDelivrпјҡжӣҫз»Ҹзҡ„зҘһпјҢзҺ°еңЁеӣҪеҶ…и®ҝй—®з»ҸеёёжҢӮжҺүпјҢдё”жұЎжҹ“ Commit и®°еҪ•гҖӮ
  • е…¬е…ұе…Қиҙ№еӣҫеәҠпјҲеҰӮ SM.MSгҖҒи·ҜиҝҮеӣҫеәҠзӯүпјүпјҡйҡҸж—¶еҸҜиғҪи·‘и·ҜпјҢж•°жҚ®ж— д»·пјҢдёҚж•ўиөҢгҖӮ
  • еӣҪеҶ…дә‘еҺӮе•Ҷ (OSS/COS)пјҡйҖҹеәҰеҝ«дҪҶеҝ…йЎ»еӨҮжЎҲпјҢдё”жөҒйҮҸиҙ№жҳҜдёӘж— еә•жҙһпјҢиў«дәәеҲ·дёҖдёӢдёҖеӨңеӣһеҲ°и§Јж”ҫеүҚгҖӮеҜ№дёӘдәәеҚҡе®ўжқҘиҜҙпјҢеӨҮжЎҲжөҒзЁӢз№ҒзҗҗгҖӮ

Cloudflare R2 жҳҜзӣ®еүҚзҡ„з»ҲжһҒзӯ”жЎҲпјҡ

зңҹжҳҜиөӣеҚҡжҙ»дҪӣгҖӮгҖӮгҖӮ

  1. е…Қиҙ№йўқеәҰеӨ§пјҡ10GB еӯҳеӮЁ + 1000 дёҮж¬Ў/жңҲиҜ»еҸ–пјҢдёӘдәәеҚҡе®ўз”ЁдёҖиҫҲеӯҗйғҪеӨҹдәҶгҖӮ
  2. е…ҚеӨҮжЎҲпјҡеӣҪйҷ…еӨ§еҺӮпјҢж— йңҖ ICP еӨҮжЎҲпјҢиҮіе°‘дёҚз”ЁжӢ…еҝғзӘҒз„¶и·‘и·ҜжҲ–еҲ еӣҫгҖӮ
  3. йӣ¶жөҒйҮҸиҙ№пјҡиҝҷжҳҜжңҖйӘҡзҡ„пјҢR2 зҡ„еҮәеҸЈжөҒйҮҸе®Ңе…Ёе…Қиҙ№пјҲеҜ№жҜ” AWS S3 зҡ„й«ҳжҳӮжөҒйҮҸиҙ№пјүгҖӮ
  4. е…је®№ S3 APIпјҡй…ҚеҗҲ PicGo/PicList жҜ«ж— еҺӢеҠӣгҖӮ

жң¬ж–Үе°ҶеёҰдҪ жҗӯе»әдёҖеҘ— вҖңйҳІзӣ—й“ҫ + е…Ёзҗғ CDN еҠ йҖҹ + иҮӘеҠЁеҺӢзј© + йӣ¶жҲҗжң¬вҖқ зҡ„еӣҫеәҠгҖӮ

з¬¬дёҖжӯҘпјҡеҲӣе»ә R2 еӯҳеӮЁжЎ¶

  1. жіЁеҶҢ/зҷ»еҪ•пјҡиҝӣе…Ҙ Cloudflare DashboardгҖӮ

  2. иҝӣе…Ҙ R2пјҡзӮ№еҮ»е·Ұдҫ§иҸңеҚ•зҡ„ R2гҖӮеҰӮжһңжҳҜз¬¬дёҖж¬ЎдҪҝз”ЁпјҢйңҖиҰҒз»‘е®ҡдҝЎз”ЁеҚЎ/PayPal ејҖйҖҡпјҲд»…йӘҢиҜҒпјҢдёҚжүЈиҙ№пјүгҖӮ

  3. еҲӣе»әеӯҳеӮЁжЎ¶ (е…ій”®жҠҖе·§)

    • зӮ№еҮ» Create bucketгҖӮ

    • еҢәеҹҹйҖүжӢ©пјҢR2 дјҡж №жҚ®дҪ зӮ№еҮ»еҲӣе»әж—¶зҡ„ IP иҮӘеҠЁеҲҶй…ҚеҢәеҹҹгҖӮ

      • жҢӮзҫҺеӣҪжўҜеӯҗ -> ж•°жҚ®иҗҪең°еҢ—зҫҺгҖӮ
      • жҢӮж—Ҙжң¬/ж–°еҠ еқЎ/йҰҷжёҜжўҜеӯҗ -> ж•°жҚ®иҗҪең°дәҡеӨӘ (жҺЁиҚҗпјҢеӣҪеҶ…и®ҝй—®жӣҙеҝ«)гҖӮ

    • з»ҷжЎ¶иө·дёӘеҗҚеӯ—пјҢжҜ”еҰӮ blog-imgгҖӮ

  4. иҺ·еҸ– API еҜҶй’Ҙ

    • еӣһеҲ° R2 дё»йЎөпјҢзӮ№еҮ»еҸідҫ§дёӢж–№зҡ„ ManageгҖӮ

    • зӮ№еҮ» Create API tokenгҖӮ

    • Permissions: йҖүжӢ© Admin Read & Write (з®ЎзҗҶе‘ҳиҜ»еҶҷ)гҖӮ

    • зӮ№еҮ»еҲӣе»әпјҢеҠЎеҝ…дҝқеӯҳеҘҪд»ҘдёӢдёүдёӘдҝЎжҒҜ

      пјҲеҸӘжҳҫзӨәдёҖж¬Ўпјүпјҡ

      • Access Key ID
      • Secret Access Key
      • S3 API Endpoint (зұ»дјјдәҺ https://<account_id>.r2.cloudflarestorage.com)

  5. еӣһеҲ°еӮЁеӯҳжЎ¶з®ЎзҗҶйЎөйқўпјҢйҡҸдҫҝдёҠдј дёҖеј жөӢиҜ•еӣҫзүҮ test.pngпјҢд»ҘдҫӣеҗҺйқўжөӢиҜ•дҪҝз”Ё

з¬¬дәҢжӯҘпјҡйғЁзҪІ Worker еҸҚд»Ј

R2 еҜ№иұЎеӮЁеӯҳзҡ„дёӘдәәе…Қиҙ№йўқеәҰжҳҜжңүйҷҗзҡ„пјҢдёәдәҶйҳІжӯў DDoS / CC зӯүжҒ¶ж„Ҹж¶ҲиҖ—жөҒйҮҸзҡ„ж”»еҮ»пјҢжңүеҝ…иҰҒдҪҝз”Ё Cloudflare Workers еҸҚд»Ј R2 еҜ№иұЎеӮЁеӯҳпјҢиҖҢдёҚжҳҜзӣҙжҺҘе°Ҷ R2 еӮЁеӯҳең°еқҖеҜ№еӨ–е…¬ејҖпјҢе®һзҺ°иҮӘе®ҡд№үеҹҹеҗҚе’ҢйҳІзӣ—й“ҫгҖӮ

  1. еҲӣе»ә Worker
    • Cloudflare еҗҺеҸ° -> Workers & Pages -> Create application -> Create WorkerгҖӮ
    • е‘ҪеҗҚдёә r2-image-proxyпјҢзӮ№еҮ»йғЁзҪІгҖӮ
  2. зј–иҫ‘д»Јз Ғ
    • зӮ№еҮ» Edit codeпјҢжё…з©әеҺҹжңүд»Јз ҒпјҢзІҳиҙҙд»ҘдёӢи„ҡжң¬пјҲи„ҡжң¬еҸӮиҖғиҮӘ Linux.DO и®әеқӣ @leonkuku еӨ§дҪ¬пјүпјҡ

  1. й…ҚзҪ®з»‘е®ҡ иҝӣе…Ҙ Worker зҡ„ Settings йЎөйқўпјҡ

    • з»‘е®ҡ R2 (Variables -> R2 Bucket Bindings)

      • зӮ№еҮ» Add bindingгҖӮ
      • Variable name: R2_BUCKET (еҝ…йЎ»е®Ңе…ЁдёҖиҮҙ)гҖӮ
      • R2 Bucket: йҖүжӢ©дҪ еҲҡжүҚеҲӣе»әзҡ„ blog-img жЎ¶гҖӮ

    • и®ҫзҪ®йҳІзӣ—й“ҫ (Variables -> Environment Variables)

      • зӮ№еҮ» Add variableгҖӮ
      • Variable name: ALLOWED_DOMAINS_STRгҖӮ
      • Value: дҪ зҡ„еҚҡе®ўеҹҹеҗҚпјҢеҰӮ https://blog.example.com,http://localhost:4000 (жң¬ең°и°ғиҜ•и®°еҫ—еҠ  localhost)гҖӮ
      • еҰӮжһңдёҚеЎ«иҝҷдёӘеҸҳйҮҸпјҢеҲҷдёҚеҗҜз”ЁйҳІзӣ—й“ҫгҖӮ

  2. з»‘е®ҡиҮӘе®ҡд№үеҹҹеҗҚпјҡ

    • Settings -> Domains & Routes -> ADDгҖӮ

    • ж·»еҠ дёҖдёӘеӯҗеҹҹеҗҚпјҢдҫӢеҰӮ img.example.comгҖӮCloudflare дјҡиҮӘеҠЁеӨ„зҗҶ DNSгҖӮ

    • жңҖеҗҺи®ҝй—®дёҖдёӢimg.example.com/test.pngиғҪжҲҗеҠҹеҮәзҺ°е°ұеҘҪ

з¬¬дёүжӯҘпјҡй…ҚзҪ® Cache Rules

жөҸи§ҲеҷЁзј“еӯҳ = еӯҳеңЁз”ЁжҲ·зҡ„з”өи„‘/жүӢжңәйҮҢгҖӮ (зңҒз”ЁжҲ·зҡ„жөҒйҮҸпјҢйҖҹеәҰжңҖеҝ«пјҢдҪҶдҪ еҫҲйҡҫжҺ§еҲ¶и®©е®ғејәеҲ¶иҝҮжңҹ)

иҫ№зјҳзј“еӯҳ = еӯҳеңЁ Cloudflare зҡ„е…ЁзҗғжңәжҲҝйҮҢгҖӮ (зңҒдҪ зҡ„жңҚеҠЎеҷЁжөҒйҮҸпјҢйҖҹеәҰеҫҲеҝ«пјҢдҪ еҸҜд»ҘйҡҸж—¶еңЁ Cloudflare еҗҺеҸ°дёҖй”®жё…йҷӨзј“еӯҳ)

Cloudflare R2 жҜҸжңҲжңү 1000W е…Қиҙ№иҜ»еҸ–ж¬Ўж•°гҖӮеҰӮжһңдҪ дёҚж”ҫеҝғпјҢеҸҜд»ҘеҶҚеўһеҠ дёҖеұӮзј“еӯҳпјҢж—ўеҸҜд»ҘеҮҸе°‘еӣһжәҗиҜ»еҸ–ж¬Ўж•°пјҢеҸҲиғҪеҠ йҖҹи®ҝй—®йҖҹеәҰгҖӮдёҚиҝҮзј“еӯҳеҸҜиғҪеҜјиҮҙеӣҫзүҮдҝ®ж”№д»ҘеҗҺжӣҙж–°дёҚеҸҠж—¶пјҢжңүиҝҷж–№йқўйңҖжұӮзҡ„еҸҜд»ҘдёҚеҗҜз”ЁжӯӨеҠҹиғҪгҖӮ

еҰӮжһңдҪ жІЎжңүз»Ҹеёёжӣҙж–°еӣҫзүҮ/ж–Үз« зҡ„йңҖжұӮпјҢйӮЈд№ҲдёәдәҶи®©еӣҫзүҮеҠ иҪҪйЈһеҝ«е№¶иҠӮзңҒ Worker/R2 зҡ„и°ғз”Ёж¬Ўж•°пјҢжҲ‘д»¬йңҖиҰҒй…ҚзҪ® Cloudflare зҡ„зј“еӯҳгҖӮ

  1. иҝӣе…ҘеҹҹеҗҚз®ЎзҗҶ -> Caching -> Cache Rules -> Create ruleгҖӮ

  2. й…ҚзҪ®еҰӮдёӢ

    • Rule Name: R2 Images Cache
    • Field: Hostname equals img.example.com (дҪ зҡ„еӣҫеәҠеҹҹеҗҚ)
    • Cache eligibility: Eligible for cache
    • Edge TTL (жңҚеҠЎеҷЁзј“еӯҳ): Ignore cache-control... -> 1 Month (з”ҡиҮі 1 Year)
    • Browser TTL (жөҸи§ҲеҷЁзј“еӯҳ): Override origin... -> 1 Month
    • Cache Key: дҝқжҢҒй»ҳи®Өзҡ„ Standard (дёҚиҰҒйҖү Ignore Query String)гҖӮ

  3. йғЁзҪІгҖӮ

  4. й…ҚзҪ®е®ҢжҲҗеҗҺзӮ№еҮ»еҸідёӢи§’йғЁзҪІпјҢзЁҚзӯүдёҖдјҡеҗҺзј“еӯҳе°ұе°Ҷз”ҹж•Ҳ

    еҗҜз”Ёзј“еӯҳеҗҺпјҢжҜҸеҪ“еӣҫеәҠзҡ„жҹҗеј еӣҫзүҮиў«и®ҝй—®пјҢCloudflare дјҡжҠҠе®ғж”ҫе…Ҙе…Ёзҗғ CDN зј“еӯҳ 24 е°Ҹж—¶пјҢ24 е°Ҹж—¶д»ҘеҶ…еҜ№иҝҷеј еӣҫзүҮзҡ„д»»ж„Ҹи®ҝй—®йғҪе°ҶзӣҙжҺҘд»Һзј“еӯҳиҜ»еҸ–гҖӮиҝҷж ·еӨ§еӨ§йҷҚдҪҺдәҶ Workers е’Ң R2 еӮЁеӯҳзҡ„иҜ»еҸ–иҜ·жұӮејҖй”ҖпјҢд№ҹжҸҗй«ҳдәҶе“Қеә”йҖҹеәҰ

  5. жҜҸж¬Ўжӣҙж–°ж–Үз« еҗҺпјҢи®°еҫ—жқҘеҹҹеҗҚз®ЎзҗҶ -> е·Ұдҫ§ж ҸжүҫеҲ°Caching -> Configuration -> Purge Everythingжё…з©әдёҖдёӢзј“еӯҳ

з¬¬еӣӣжӯҘпјҡй…ҚзҪ® PicList е®ўжҲ·з«Ҝ

жң¬жқҘжҲ‘жҳҜиҰҒз”ЁPicGoзҡ„пјҢжҜ•з«ҹд№ӢеүҚе°ұеңЁз”ЁпјҢдҪҶжҳҜ==еҘҪеҗ§пјҢжҲ‘й…ҚзҪ®зҡ„ж—¶еҖҷеҸ‘зҺ°жІЎжңүиҮӘе®ҡд№үеҹҹеҗҚзҡ„йҖүйЎ№пјҢжҚўPicList==

жҺЁиҚҗдҪҝз”Ё PicList иҖҢдёҚжҳҜ PicGoпјҢе®ғжҳҜ PicGo зҡ„ж”№зүҲпјҢз»ҙжҠӨжӣҙз§ҜжһҒпјҢеҠҹиғҪжӣҙејәгҖӮ

  1. дёӢиҪҪе®үиЈ…пјҡPicList Github ReleaseгҖӮ
  2. й…ҚзҪ® S3 еӣҫеәҠ
    • жү“ејҖ PicList -> еӣҫеәҠи®ҫзҪ® -> Amazon S3гҖӮ
    • й…ҚзҪ®еҗҚ: иҮӘе·ұиө·дёҖдёӘеҗҚеӯ—пјҲжІЎй”ҷжҲ‘жҳҜиө·еҗҚ/йҖүжӢ©еӣ°йҡҫз—Үпјү
    • AccessKeyId: з¬¬дёҖжӯҘиҺ·еҸ–зҡ„ Key IDгҖӮ
    • SecretAccessKey: з¬¬дёҖжӯҘиҺ·еҸ–зҡ„ Secret KeyгҖӮ
    • Bucket: blog-img (дҪ зҡ„жЎ¶еҗҚ)гҖӮ
    • Region: auto (жҲ–иҖ… apac еҰӮжһңдҪ еңЁдәҡеӨӘ)гҖӮ
    • иҮӘе®ҡд№үиҠӮзӮ№: з¬¬дёҖжӯҘиҺ·еҸ–зҡ„ S3 API Endpoint (жіЁж„ҸпјҡдёҚиҰҒеёҰжЎ¶еҗҚпјҢж јејҸеҰӮ https://xxx.r2.cloudflarestorage.com)гҖӮ
    • иҮӘе®ҡд№үеҹҹеҗҚ: https://img.example.com (дҪ зҡ„ Worker еҹҹеҗҚ)гҖӮ
  3. и®ҫдёәй»ҳи®ӨеӣҫеәҠгҖӮ

иҝӣйҳ¶пјҡеӣҫзүҮиҮӘеҠЁеҺӢзј© (еҸҜйҖү)

PicList ж”ҜжҢҒдёҠдј еүҚиҮӘеҠЁеҺӢзј©пјҢејәзғҲжҺЁиҚҗй…ҚзҪ®гҖӮ

  1. е®үиЈ…жҸ’д»¶picgo-plugin-compress-next

    • PicList иҪҜд»¶еҶ… -> жҸ’д»¶и®ҫзҪ® -> жҗңзҙў picgo-plugin-compress-next е®үиЈ…гҖӮ

    • еҰӮжһңе®үиЈ…ж…ўпјҡиҝӣе…Ҙ PicList е®үиЈ…зӣ®еҪ•пјҢиҝҗиЎҢ npm i picgo-plugin-compress-nextгҖӮ

  2. й…ҚзҪ®жҸ’д»¶

    • зӮ№еҮ»жҸ’д»¶зҡ„е°ҸйҪҝиҪ®еӣҫж ҮгҖӮ
    • жҺЁиҚҗз»„еҗҲпјҡTinyPNG (йңҖз”іиҜ·API Key)) + webp-converter (жң¬ең°иҪ¬жҚў)гҖӮе…·дҪ“иҜ·зңӢжҸ’д»¶GithubйЎө
    • ејҖеҗҜ transformer - compress-nextгҖӮ
    • иҝҷж ·дёҠдј зҡ„еӣҫзүҮдјҡиҮӘеҠЁиў«еҺӢзј©пјҢдҪ“з§ҜеҮҸе°Ҹ 50% д»ҘдёҠпјҢдёӢйқўжҺҘзқҖз”ЁPicListиҮӘеёҰзҡ„еҺӢзј©+еӣҫзүҮиҪ¬жҚўиғҪеҶҚиҝӣжӯҘ

  3. иҮӘеёҰеҺӢзј©

    • еӣһеҲ°еӣҫзүҮдёҠдј йЎөйқўпјҢжңүйңҖиҰҒзҡ„иҜқеҸҜд»Ҙи®ҫзҪ®дёҖдёӢеҸідёҠи§’зҡ„еӣҫзүҮеӨ„зҗҶи®ҫзҪ®
    • иҝҷйҮҢйҷӨдәҶжҸ’д»¶еҺӢзј©пјҢеүҚйқўжҲ‘д»¬з”Ёзҡ„жҳҜTinyPng+webp-converterпјҢжҲ‘д»¬иҝҳеҸҜд»Ҙз”ЁPicListеҶҚеҺӢзј©+иҪ¬жҚўдёҖж¬Ў

    ж°ҙеҚ°е°ұдёҚеҠ дәҶгҖӮгҖӮгҖӮ

з¬¬дә”жӯҘпјҡж”¶е°ҫ

  1. Typora и®ҫзҪ®

    • Typora -> еҒҸеҘҪи®ҫзҪ® -> еӣҫеғҸ -> дёҠдј жңҚеҠЎ -> йҖүжӢ© PicList (жіЁж„ҸдёҚжҳҜ PicGo-Core)гҖӮ
    • йӘҢиҜҒеӣҫзүҮдёҠдј йҖүйЎ№гҖӮ

  2. е…ій—ӯ r2.dev и®ҝй—®

    • еӣһеҲ° Cloudflare R2 еӯҳеӮЁжЎ¶и®ҫзҪ® -> SettingsгҖӮ
    • жүҫеҲ° Public access -> R2.dev subdomainгҖӮ
    • зЎ®дҝқе®ғжҳҜ Disallow Access зҠ¶жҖҒгҖӮ
    • зҗҶз”ұпјҡжҲ‘д»¬е·Із»ҸжңүдәҶиҮӘе®ҡд№үеҹҹеҗҚ WorkerпјҢдёҚйңҖиҰҒиҝҷдёӘе…¬ејҖзҡ„жөӢиҜ•еҹҹеҗҚпјҢйҳІжӯўиў«з»•иҝҮйҳІзӣ—й“ҫгҖӮ
