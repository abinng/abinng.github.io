жң¬зҜҮдё»иҰҒдәҶи§ЈеҚіеҸҜпјҢеҲқеӯҰж—¶дёҚж·ұе…ҘдәҶи§Јд№ҹиЎҢ

еј•е…Ҙ

Qt зҡ„е…ғеҜ№иұЎдҪҝз”Ёзү№жҖ§

  • зұ»дёӯеҶҷ Q_OBJECT е®Ҹ
  • з»§жүҝ QObject
  • дёҚиғҪж”ҫеҲ° cpp ж–Үд»¶дёӯ

еүҚдёӨдёӘзҡ„ж•Ҳжһңе°ұеҸҜд»ҘзҗҶи§ЈжҲҗпјҢжҷ®йҖҡзҡ„зұ»з»“жһ„еӨ–жү©дәҶдёҖйғЁеҲҶпјҢе°ұеҸ«еҒҡе…ғеҜ№иұЎпјҢе°ұжҳҜдёҖдёӘз©әй—ҙпјҢеҢ…еҗ«з»“жһ„дҝЎжҒҜ

иҖҢжҲ‘д»¬зҡ„ Qt Core иҰҒи§ЈжһҗиҝҷйғЁеҲҶеӨ–жү©зҡ„з»“жһ„пјҢйңҖиҰҒдёҖдёӘAPIжҺҘеҸЈпјҲQt е·Із»Ҹе®һзҺ°еҘҪзҡ„пјүпјҢжүҖд»Ҙе…¶дёӯзҡ„з»“жһ„д№ҹиҰҒжҢүз…§ Qt иҰҒжұӮзҡ„з»“жһ„е®һзҺ°

еҺҹжқҘзҡ„йӮЈдёҖйғЁеҲҶе°ұдёҚйңҖиҰҒи§ЈжһҗпјҢиҝҷжҳҜжҲ‘д»¬иҮӘе·ұе®ҡд№үзҡ„жҲҗе‘ҳ

йӮЈдёәд»Җд№ҲдёҚиғҪж”ҫеҲ° cpp ж–Үд»¶дёӯе‘ўпјҹ

зұ»зҡ„е®ҡд№үиӮҜе®ҡжҳҜеңЁ .h ж–Үд»¶дёӯпјҢеҜ№еә”зҡ„еҮҪж•°е®һзҺ°еңЁ .cpp дёӯпјҢAPI жҺҘеҸЈе°ұеҸҜд»Ҙи°ғз”ЁеҮҪж•°е®һзҺ°пјҢжқҘжҸҗеҸ–иҝҷйғЁеҲҶеӨ–жү©зҡ„з»“жһ„дҝЎжҒҜгҖӮиҝҷйғЁеҲҶеҮҪж•°е®һзҺ°д№ҹжҳҜ Qt з”ҹжҲҗзҡ„

д№ӢеүҚдәҶи§ЈиҝҮ UIC е’Ң RCC

UICпјҡе°Ҷ UI ж–Үд»¶иҪ¬жҚўжҲҗ .h ж–Үд»¶пјҢдјҡиў«жү“еҢ…еҲ°еҸҜжү§иЎҢж–Үд»¶дёӯ RCCпјҡе°Ҷиө„жәҗж–Үд»¶жү“еҢ…жҲҗ cpp ж–Үд»¶пјҢдјҡиў«жү“еҢ…еҲ°еҸҜжү§иЎҢж–Үд»¶дёӯ

иҖҢзҺ°еңЁиҝҷдёӘе…ғеҜ№иұЎзі»з»ҹеҸҜд»Ҙз§°дёә MOC (Meta-ObjectВ Compiler)пјҢжҠҠеҪ“еүҚзӣ®еҪ•дёӢжүҖжңүеҢ…еҗ«еӨ–жү©йғЁеҲҶзҡ„еӨҙж–Үд»¶пјҢе°ҶеҮҪж•°е®һзҺ°з”ҹжҲҗеҲ°дёҖдёӘ cpp ж–Үд»¶дёӯпјҢжңҖеҗҺд№ҹдјҡиў«жү“еҢ…еҲ°еҸҜжү§иЎҢж–Үд»¶дёӯ

е…¶е®һиҜҘиҝҮзЁӢе°ұжҳҜ 001-Qt зҡ„зј–иҜ‘еҺҹзҗҶ дёӯйӮЈе№…еӣҫ

MOC дё»иҰҒжҳҜдёәдәҶдҝқиҜҒиҝҗиЎҢж—¶пјҢеҸҜд»ҘеҠЁжҖҒи°ғж•ҙеҜ№иұЎзҡ„еұһжҖ§еҖјгҖӮзӣёеҪ“дәҺдёҖдёӘеҸҚе°„зҡ„жҖқжғігҖӮ

дҫӢеҰӮпјҡи®ҫи®ЎдёҖдёӘжёёжҲҸзҡ„жңәеҲ¶пјҢеҲҡејҖе§Ӣи®ҫи®Ўж—¶пјҢеҸҜиғҪеҸӘиҖғиҷ‘еҲ°дәҶжҺүиЎҖпјҢдҪҶеҗҺйқўйҖҗжёҗжү©е……жёёжҲҸзҺ©жі•пјҢжңүзҡ„ж”»еҮ»еҸҜиғҪдјҡжҺүиЎҖпјҢжңүзҡ„ж”»еҮ»еҸҜиғҪдјҡжҺүйӯ”жі•еҖјпјҢжҲ–иҖ…е…¶д»–еҖјпјҢжӯӨж—¶иҰҒйҮҚж–°и®ҫи®ЎжЎҶжһ¶пјҢе°ұдјҡеҫҲеӨҚжқӮгҖӮеҸӘйңҖиҰҒзңӢж”»еҮ»зҡ„жӯҰеҷЁйңҖиҰҒеҮҸд»Җд№Ҳзұ»еһӢзҡ„ж•°еҖјпјҢеҜ№еә”еҸ—еҮ»иӢұйӣ„жңүжІЎжңүеҜ№еә”зҡ„еұһжҖ§пјҢжқҘжү§иЎҢеҜ№еә”зҡ„ж“ҚдҪңжҲ–иҖ…е…¶д»–ж“ҚдҪңе°ұеҘҪгҖӮиҝҷд№ҲдёҖжқҘе°ұи§ЈиҖҰдәҶ

йҖҡиҝҮдёҖдёӘеӯ—з¬ҰдёІжқҘжҹҘжүҫжҳҜеҗҰжңүе“Қеә”зҡ„еұһжҖ§пјҢжқҘжү§иЎҢзӣёе…іеҠЁдҪң

з”ұжӯӨеј•еҮәдҝЎеҸ·е’Ңж§ҪпјҢе…¶жң¬иә«и®ҫи®ЎеҫҲж…ўпјҢдҪҶеҫҲж–№дҫҝпјҢдҝЎеҸ·еҸ‘еҮәиҖ…е’ҢдҝЎеҸ·еӨ„зҗҶиҖ…д№Ӣй—ҙи§ЈиҖҰпјҢз”ұз¬¬дёүж–№жқҘеҶіе®ҡи°ҒеҜ№еә”и°ҒгҖӮ

з”ұдәҺеӨ–жү©дәҶйғЁеҲҶз»“жһ„дҝЎжҒҜпјҢиҝҳеўһеҠ дәҶжҷәиғҪйҖүжӢ©еӨ„зҗҶеҮҪж•°зҡ„иҝҮзЁӢпјҢж—¶й—ҙе’Ңз©әй—ҙйғҪеўһеҠ дәҶ

дёҺд№ӢеҜ№еә”зҡ„жҳҜеӣһи°ғеҮҪж•°зҡ„жғ…еҶөпјҢи§ҰеҸ‘дәҶжҹҗжҹҗжқЎд»¶пјҢе°ұзӣҙжҺҘи°ғз”ЁеҜ№еә”зҡ„еҮҪж•°пјҢзӣҙжҺҘз»‘е®ҡдәҶпјҢзӣёеҪ“дәҺеҫ—жҸҗеүҚзҹҘйҒ“еҜ№еә”е…ізі»пјҢжҜ”иҫғдёҚзҒөжҙ»

йҰ–е…Ҳи®ІдёҖдёӢжІЎи§ҒиҝҮзҡ„е®Ҹпјҡ

ж„ҸжҖқжҳҜпјҢMonster зұ»е‘ҠиҜү Qt пјҡ

  • еҗҚеӯ—пјҡжҲ‘жңүдёҖдёӘеұһжҖ§еҸ« healthгҖӮ
  • зұ»еһӢпјҡе®ғзҡ„зұ»еһӢжҳҜ intгҖӮ
  • з»‘е®ҡпјҡеҪ“дҪ пјҲQt зі»з»ҹпјүжғіиҰҒиҜ»еҶҷиҝҷдёӘеұһжҖ§ж—¶пјҢиҜ·зӣҙжҺҘж“ҚдҪңжҲ‘зұ»йҮҢзҡ„жҲҗе‘ҳеҸҳйҮҸ m_healthпјҲиҝҷе°ұжҳҜ MEMBER е…ій”®еӯ—зҡ„дҪңз”ЁпјүгҖӮ

MOC е№ІдәҶд»Җд№Ҳпјҡ

еҪ“дҪ зӮ№еҮ»зј–иҜ‘ж—¶пјҢQt зҡ„ MOCпјҲе…ғеҜ№иұЎзј–иҜ‘еҷЁпјү дјҡжү«жҸҸдҪ зҡ„еӨҙж–Үд»¶пјҡ

  1. з”ҹжҲҗе…ғж•°жҚ®пјҡMOC дјҡз”ҹжҲҗдёҖдёӘйҡҗи—Ҹзҡ„ C++ ж–Үд»¶пјҲйҖҡеёёеҸ« moc_monster.cppпјүгҖӮ
  2. зҷ»и®°иЎЁпјҡеңЁиҝҷдёӘйҡҗи—Ҹж–Үд»¶йҮҢпјҢMOC з»ҙжҠӨдәҶдёҖеј е·ЁеӨ§зҡ„иЎЁпјҢи®°еҪ•дәҶ Monster зұ»йҮҢжңүдёҖдёӘеҸ« "health" зҡ„еӯ—з¬ҰдёІпјҢеҜ№еә”зҡ„жҳҜ int зұ»еһӢгҖӮ
  3. еҸҚе°„жңәеҲ¶пјҡе®ғе®һзҺ°дәҶвҖңеҸҚе°„вҖқпјҲReflectionпјүгҖӮиҝҷж„Ҹе‘ізқҖдҪ еҸҜд»ҘеңЁиҝҗиЎҢж—¶йҖҡиҝҮеӯ—з¬ҰдёІжқҘж“ҚдҪңеҸҳйҮҸгҖӮ

з„¶еҗҺжҲ‘д»¬еҶҚжқҘеҜ№еә”зқҖд»Јз ҒзңӢеӣҫпјҡ

д»Јз ҒжҳҜжҖҺд№ҲеҚ°иҜҒжһ¶жһ„еӣҫзҡ„е‘ўпјҹ

  1. и“қеҢәпјҲдёҚйңҖиҰҒи§Јжһҗзҡ„жҷ®йҖҡжҲҗе‘ҳпјүпјҡ еҜ№еә” Monster зұ»йҮҢзҡ„з§ҒжңүеҸҳйҮҸ m_health е’Ң m_manaгҖӮе®ғд»¬жҳҜ C++ еҺҹз”ҹзҡ„пјҢи—ҸеңЁй»‘зӣ’йҮҢгҖӮ

  2. зәўеҢәпјҲе…ғеҜ№иұЎз©әй—ҙ з»“жһ„дҝЎжҒҜпјүпјҡ еҜ№еә” Q_PROPERTY(int health MEMBER m_health)гҖӮиҝҷзӣёеҪ“дәҺеңЁ Qt зҡ„вҖңжҲ·еҸЈжң¬вҖқдёҠеҒҡдәҶзҷ»и®°пјҢжҡҙйңІдәҶдёҖдёӘеҜ№еӨ–зҡ„еӯ—з¬ҰдёІеҗҚеӯ— "health"гҖӮ

  3. API жҺҘеҸЈдёҺеҸҚе°„жү§иЎҢпјҲж ёеҝғжЁЎжӢҹпјүпјҡ

    • жҲ‘д»¬зңӢ lost еҮҪж•°пјҢе®ғе®Ңе…ЁдёҚзҹҘйҒ“дј е…Ҙзҡ„еҜ№иұЎжҳҜ Monster иҝҳжҳҜе…¶д»–д»Җд№ҲжҖӘзү©пјҢе®ғеҸӘи®Ө QObject жҢҮй’ҲгҖӮ

    • obj->metaObject()пјҡиҝҷе°ұжҳҜиҺ·еҸ–еӣҫдёӯзҡ„ вҖңAPI жҺҘеҸЈвҖқпјҢжӢҝеҲ°дәҶиҝҷдёӘеҜ№иұЎзҡ„е…ғдҝЎжҒҜеҗҚеҶҢгҖӮ

    • meta->indexOfProperty(propertyName)пјҡиҝҷе°ұжҳҜеӣҫжіЁйҮҢеҶҷзҡ„ вҖңйҖҡиҝҮеӯ—з¬ҰдёІжқҘжҹҘжүҫжҳҜеҗҰжңүеұһжҖ§вҖқгҖӮдј е…Ҙ "health"пјҢе®ғе°ұеҺ»зәўеҢәйҮҢжүҫжңүжІЎжңүиҝҷдёӘеҗҚеӯ—гҖӮ

    • property.read() / property.write()пјҡдёҖж—ҰжүҫеҲ°пјҢдёҚйңҖиҰҒжүӢеҠЁи°ғз”Ё mon1.m_healthпјҢзӣҙжҺҘйҖҡиҝҮ Qt Core жҸҗдҫӣзҡ„ API жҺҘеҸЈеҠЁжҖҒдҝ®ж”№дәҶеә•еұӮж•°жҚ®гҖӮ

иҝҷж®өд»Јз Ғзҡ„ж„Ҹд№үпјҲдёәд»Җд№ҲиҜҙе®ғи§ЈиҖҰдәҶпјҹпјүпјҡ жіЁж„ҸзңӢ lost(QObject *obj, const char *propertyName) иҝҷдёӘеҮҪж•°гҖӮеҒҮи®ҫд»ҘеҗҺеўһеҠ дәҶдёҖдёӘ Hero зұ»пјҢеҸӘиҰҒе®ғд№ҹжңү "health" иҝҷдёӘ Q_PROPERTYпјҢдҪ дёҚйңҖиҰҒж”№еҠЁд»»дҪ•йҖ»иҫ‘пјҢзӣҙжҺҘжҠҠ Hero дј з»ҷ lost еҮҪж•°пјҢз…§ж ·иғҪжүЈиЎҖпјҒиҝҷе°ұжҳҜйқўеҗ‘еӯ—з¬ҰдёІзј–зЁӢпјҲеҸҚе°„пјүеёҰжқҘзҡ„жһҒеӨ§зҒөжҙ»жҖ§гҖӮ
