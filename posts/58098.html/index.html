дҝқеӯҳйҖҖеҮә

иҝҷйҮҢй»ҳи®ӨдҪ е·Із»ҸжңүgccдәҶпјҢеҰӮжһңжІЎжңүпјҢemmвҖҰ.еҺ»иЈ…дёҖдёӢ

Linux

иҝҷж®өиҫ“еҮәжҳҜдҪ иҝҗиЎҢВ gcc -v -o build hello.cВ е‘Ҫд»Өдә§з”ҹзҡ„з»“жһңгҖӮ

  • -v: иЎЁзӨәВ VerboseпјҲиҜҰз»ҶжЁЎејҸпјүпјҢи®© GCC жү“еҚ°еҮәе®ғеңЁзј–иҜ‘иҝҮзЁӢдёӯжү§иЎҢзҡ„жүҖжңүе…·дҪ“жӯҘйӘӨе’Ңй…ҚзҪ®дҝЎжҒҜгҖӮ
  • -o build: жҢҮе®ҡиҫ“еҮәзҡ„еҸҜжү§иЎҢж–Үд»¶еҗҚдёәВ buildгҖӮ
  • hello.c: жәҗж–Үд»¶гҖӮ

GCCпјҲGNU Compiler Collectionпјүе®һйҷ…дёҠжҳҜдёҖдёӘй©ұеҠЁзЁӢеәҸпјҲDriverпјүпјҢе®ғе№¶дёҚзӣҙжҺҘе№Іжҙ»пјҢиҖҢжҳҜжҢҮжҢҘе…¶д»–зЁӢеәҸпјҲйў„еӨ„зҗҶеҷЁгҖҒзј–иҜ‘еҷЁгҖҒжұҮзј–еҷЁгҖҒй“ҫжҺҘеҷЁпјүжқҘе®ҢжҲҗе·ҘдҪңгҖӮ

жҲ‘д»¬еҸҜд»ҘжҠҠиҝҷж®өеӨҚжқӮзҡ„иҫ“еҮәеҲҶдёәеӣӣдёӘдё»иҰҒйғЁеҲҶжқҘи§ЈиҜ»пјҡ

з¬¬дёҖйғЁеҲҶпјҡGCC зҡ„й…ҚзҪ®дҝЎжҒҜ (Configuration)

д»ҺејҖеӨҙеҲ°В gcc version ...В иҝҷдёҖеӨ§ж®өгҖӮ

  • Target:В x86_64-linux-gnuВ иЎЁзӨәзӣ®ж Үжһ¶жһ„жҳҜ 64дҪҚ LinuxгҖӮ
  • Configured with: иҝҷйҮҢеҲ—еҮәдәҶеҪ“еҲқзј–иҜ‘е®үиЈ…иҝҷдёӘ GCC зүҲжң¬ж—¶жүҖдҪҝз”Ёзҡ„жүҖжңүеҸӮж•°пјҲжҜ”еҰӮж”ҜжҢҒе“ӘдәӣиҜӯиЁҖгҖҒе®үиЈ…и·Ҝеҫ„еңЁе“ӘйҮҢгҖҒеҗҜз”ЁдәҶе“Әдәӣзү№жҖ§зӯүпјүгҖӮ
  • gcc version: дҪ еҪ“еүҚдҪҝз”Ёзҡ„зүҲжң¬жҳҜ GCC 11.4.0гҖӮ

з¬¬дәҢйғЁеҲҶпјҡзј–иҜ‘ (Compilation) вҖ”вҖ”В cc1

иҝҷжҳҜе°Ҷ C д»Јз ҒиҪ¬жҚўдёәжұҮзј–д»Јз Ғзҡ„жӯҘйӘӨгҖӮ

  • cc1: иҝҷжҳҜзңҹжӯЈзҡ„В C зј–иҜ‘еҷЁж ёеҝғзЁӢеәҸгҖӮ

  • иҫ“е…Ҙ:В hello.cгҖӮ

  • иҫ“еҮә:В -o /tmp/ccskptD9.sгҖӮе®ғжҠҠ C д»Јз Ғзҝ»иҜ‘жҲҗдәҶжұҮзј–д»Јз ҒпјҢе№¶дёҙж—¶дҝқеӯҳеңЁВ /tmpВ зӣ®еҪ•дёӢзҡ„дёҖдёӘВ .sВ ж–Үд»¶дёӯгҖӮ

  • еӨҙж–Үд»¶жҗңзҙўи·Ҝеҫ„:
    иҫ“еҮәдёӯиҝҳжңүдёҖж®өВ #include "..." search starts here:пјҢеҲ—еҮәдәҶзј–иҜ‘еҷЁеҺ»е“ӘйҮҢжүҫеӨҙж–Үд»¶пјҲжҜ”еҰӮВ <stdio.h>пјүпјҡ

з¬¬дёүйғЁеҲҶпјҡжұҮзј– (Assembly) вҖ”вҖ”В as

иҝҷжҳҜе°ҶжұҮзј–д»Јз ҒиҪ¬жҚўдёәжңәеҷЁд»Јз ҒпјҲзӣ®ж Үж–Үд»¶пјүзҡ„жӯҘйӘӨгҖӮ

  • as: иҝҷжҳҜВ GNU AssemblerпјҲжұҮзј–еҷЁпјүгҖӮ
  • иҫ“е…Ҙ:В /tmp/ccskptD9.sпјҲдёҠдёҖжӯҘз”ҹжҲҗзҡ„жұҮзј–ж–Үд»¶пјүгҖӮ
  • иҫ“еҮә:В -o /tmp/ccyULprM.oгҖӮе®ғжҠҠжұҮзј–д»Јз Ғзҝ»иҜ‘жҲҗдәҶжңәеҷЁз ҒпјҲObject fileпјҢзӣ®ж Үж–Үд»¶пјүпјҢд№ҹжҳҜдёҙж—¶дҝқеӯҳеңЁВ /tmpВ дёӢгҖӮ

з¬¬еӣӣйғЁеҲҶпјҡй“ҫжҺҘ (Linking) вҖ”вҖ”В collect2

иҝҷжҳҜе°Ҷзӣ®ж Үж–Үд»¶е’Ңзі»з»ҹеә“з»„еҗҲжҲҗжңҖз»ҲеҸҜжү§иЎҢж–Үд»¶зҡ„жӯҘйӘӨгҖӮ

  • collect2: иҝҷжҳҜ GCC зҡ„й“ҫжҺҘеҷЁеҢ…иЈ…еҷЁпјҲйҖҡеёёе®ғеҶ…йғЁдјҡи°ғз”ЁВ ldВ й“ҫжҺҘеҷЁпјүгҖӮ
  • иҫ“е…Ҙ:
    • /tmp/ccyULprM.o: еҲҡжүҚз”ҹжҲҗзҡ„дҪ зҡ„д»Јз Ғзҡ„зӣ®ж Үж–Үд»¶гҖӮ
    • Scrt1.o,В crti.o,В crtbeginS.o,В crtendS.o,В crtn.o: иҝҷдәӣжҳҜВ C Runtime (CRT)В еҗҜеҠЁж–Үд»¶гҖӮе®ғд»¬еҢ…еҗ«дәҶзЁӢеәҸеҗҜеҠЁпјҲеңЁВ mainВ еҮҪж•°д№ӢеүҚпјүе’ҢйҖҖеҮәжүҖйңҖзҡ„еҲқе§ӢеҢ–д»Јз ҒгҖӮ
    • -lgcc,В -lgcc_s,В -lc: й“ҫжҺҘж ҮеҮҶ C еә“пјҲlibcпјүе’Ң GCC еҶ…йғЁеә“гҖӮ
  • иҫ“еҮә:В -o buildгҖӮжңҖз»Ҳз”ҹжҲҗдәҶеҗҚдёәВ buildВ зҡ„еҸҜжү§иЎҢж–Үд»¶гҖӮ

жҖ»з»“

дҪ зҡ„дёҖжқЎе‘Ҫд»ӨВ gcc hello.c -o buildВ е®һйҷ…дёҠи§ҰеҸ‘дәҶд»ҘдёӢжөҒж°ҙзәҝпјҡ

  1. cc1В (зј–иҜ‘еҷЁ):В hello.cВ (CиҜӯиЁҖ) ->В .sВ (жұҮзј–иҜӯиЁҖ)
  2. asВ (жұҮзј–еҷЁ):В .sВ (жұҮзј–иҜӯиЁҖ) ->В .oВ (жңәеҷЁз Ғ)
  3. collect2/ldВ (й“ҫжҺҘеҷЁ):В .oВ + зі»з»ҹеҗҜеҠЁж–Үд»¶ + Cж ҮеҮҶеә“ ->В buildВ (жңҖз»ҲеҸҜжү§иЎҢзЁӢеәҸ)

-vВ еҸӮж•°еҸӘжҳҜжҠҠе№•еҗҺзҡ„иҝҷдёҖеҲҮвҖңй»‘зӣ’ж“ҚдҪңвҖқжү“еҚ°з»ҷдҪ зңӢдәҶгҖӮ

Windows

иҝҷж®өиҫ“еҮәе’ҢдҪ д№ӢеүҚеңЁ Linux дёҠзңӢеҲ°зҡ„йҖ»иҫ‘жҳҜдёҖжЁЎдёҖж ·зҡ„пјҢдҪҶиҝҷеӣһжҳҜеңЁВ Windows 11В зҺҜеўғдёӢиҝҗиЎҢзҡ„гҖӮ

дҪ дҪҝз”Ёзҡ„жҳҜВ TDM-GCCпјҲGCC зҡ„дёҖдёӘ Windows з§»жӨҚзүҲжң¬пјҢеҹәдәҺ MinGW-w64пјүгҖӮ

иҷҪз„¶жөҒзЁӢиҝҳжҳҜвҖңзј–иҜ‘ -> жұҮзј– -> й“ҫжҺҘвҖқпјҢдҪҶеңЁ Windows дёҠпјҢй“ҫжҺҘпјҲLinkingпјүВ иҝҷдёҖжӯҘж¶үеҸҠзҡ„зі»з»ҹеә“иҰҒжҜ” Linux еӨҚжқӮеҫ—еӨҡгҖӮ

жҲ‘д»¬еҗҢж ·еҲҶеӣӣжӯҘжқҘи§ЈиҜ»пјҡ

з¬¬дёҖйғЁеҲҶпјҡзҺҜеўғй…ҚзҪ® (Configuration)

  • Target:В x86_64-w64-mingw32гҖӮ
    • x86_64: 64дҪҚжһ¶жһ„гҖӮ
    • w64: жҢҮеҗ‘ Windows 64дҪҚгҖӮ
    • mingw32: д»ЈиЎЁВ MinimalistВ GNU forВ WindowsгҖӮиҝҷжҳҜи®© GCC иғҪеңЁ Windows дёҠиҝҗиЎҢе№¶з”ҹжҲҗ Windows зЁӢеәҸпјҲ.exeпјүзҡ„е…ій”®е·Ҙе…·йӣҶгҖӮ
  • зүҲжң¬: дҪ дҪҝз”Ёзҡ„жҳҜ GCC 10.3.0 (TDM-GCC зүҲжң¬)гҖӮ

з¬¬дәҢйғЁеҲҶпјҡзј–иҜ‘ (Compilation) вҖ”вҖ”В cc1.exe

  • зЁӢеәҸ:В cc1.exeпјҲWindowsдёӢеӨҡдәҶ .exe еҗҺзјҖпјүгҖӮ
  • иҫ“е…Ҙ:В hello.cгҖӮ
  • иҫ“еҮә: жұҮзј–д»Јз Ғиў«иҫ“еҮәеҲ°дәҶдҪ зҡ„з”ЁжҲ·дёҙж—¶зӣ®еҪ•В C:\Users\abinng\AppData\Local\Temp\В дёӢпјҢж–Үд»¶еҗҚдёәВ ccfDrkef.sгҖӮ
  • еӨҙж–Үд»¶и·Ҝеҫ„: зј–иҜ‘еҷЁеҺ»В D:/09_SW/_Dev/TDM-GCC-64/...В дёӢйқўжүҫВ <stdio.h>В зӯүеӨҙж–Үд»¶пјҢиҖҢдёҚжҳҜ Linux дёӢзҡ„В /usr/includeгҖӮ

з¬¬дёүйғЁеҲҶпјҡжұҮзј– (Assembly) вҖ”вҖ”В as.exe

  • зЁӢеәҸ:В as.exeпјҲжұҮзј–еҷЁпјүгҖӮ
  • иҫ“е…Ҙ: еҲҡжүҚз”ҹжҲҗзҡ„дёҙж—¶жұҮзј–ж–Үд»¶В .sгҖӮ
  • иҫ“еҮә: з”ҹжҲҗдәҶжңәеҷЁз Ғзӣ®ж Үж–Үд»¶В .oпјҢд№ҹж”ҫеңЁ Temp зӣ®еҪ•дёӢгҖӮ

з¬¬еӣӣйғЁеҲҶпјҡй“ҫжҺҘ (Linking) вҖ”вҖ”В collect2.exe

иҝҷжҳҜ Linux е’Ң Windows еҢәеҲ«жңҖеӨ§зҡ„ең°ж–№гҖӮ

  • зЁӢеәҸ:В collect2.exeпјҲй“ҫжҺҘеҷЁеҢ…иЈ…еҷЁпјүгҖӮ
  • иҫ“еҮә:В build.exeпјҲWindows еҸҜжү§иЎҢж–Үд»¶пјүгҖӮ
  • й“ҫжҺҘзҡ„еә“ (Libraries):
    еңЁ Linux дёҠпјҢдҪ дё»иҰҒй“ҫжҺҘВ libcгҖӮдҪҶеңЁ Windows дёҠпјҢдёәдәҶи®©дёҖдёӘ C зЁӢеәҸиғҪи·‘иө·жқҘпјҢGCC еҝ…йЎ»й“ҫжҺҘдёҖеӨ§е Ҷ Windows зү№жңүзҡ„еә“пјҡ
    • -lmingw32,В -lmingwex: MinGW зҡ„иҝҗиЎҢж—¶еә“пјҢз”ЁжқҘеЎ«иЎҘ Linux е’Ң Windows д№Ӣй—ҙзҡ„е·®ејӮгҖӮ
    • -lmsvcrt:В MicrosoftВ VisualВ CВ RuntimeгҖӮиҝҷжҳҜ Windows зі»з»ҹиҮӘеёҰзҡ„ C ж ҮеҮҶеә“пјҲзӣёеҪ“дәҺ Linux зҡ„ glibcпјүгҖӮ
    • -lkernel32: Windows еҶ…ж ё APIпјҲеӨ„зҗҶеҶ…еӯҳгҖҒиҝӣзЁӢгҖҒж–Үд»¶IOзӯүж ёеҝғеҠҹиғҪпјүгҖӮ
    • -luser32: з”ЁжҲ·з•Ңйқў APIгҖӮ
    • -lshell32,В -ladvapi32: е…¶д»– Windows зі»з»ҹжңҚеҠЎгҖӮ
  • еҗҜеҠЁж–Үд»¶:
    • crt2.o,В crtbegin.o: Windows зЁӢеәҸзҡ„еҗҜеҠЁд»Јз ҒпјҲиҙҹиҙЈеңЁи°ғз”ЁВ main()В д№ӢеүҚеҲқе§ӢеҢ–зҺҜеўғпјүгҖӮ
    • default-manifest.o: иҝҷжҳҜдёҖдёӘ Windows зү№жңүзҡ„дёңиҘҝгҖӮе®ғеҢ…еҗ«дәҶдёҖдёӘ XML жё…еҚ•пјҢе‘ҠиҜү Windows 11 иҝҷдёӘзЁӢеәҸж”ҜжҢҒй«ҳ DPI жҳҫзӨәпјҢжҲ–иҖ…йңҖиҰҒд»Җд№Ҳж ·зҡ„жқғйҷҗгҖӮ

жҖ»з»“пјҡLinux vs Windows

жӯҘйӘӨ Linux (Ubuntu) Windows (TDM-GCC)
зј–иҜ‘еҷЁж ёеҝғ cc1 cc1.exe
и·Ҝеҫ„йЈҺж ј /usr/lib/... D:\09_SW\...
дёҙж—¶ж–Үд»¶ /tmp/... C:\Users\abinng\AppData\Local\Temp\...
жңҖз»Ҳдә§зү© ELF ж јејҸ (ж— еҗҺзјҖ) PE ж јејҸ (.exe)

з®ҖеҚ•жқҘиҜҙпјҢиҝҷж®өж—Ҙеҝ—еұ•зӨәдәҶ GCC еҰӮдҪ•еңЁ Windows дёҠвҖңе…Ҙд№ЎйҡҸдҝ—вҖқпјҢжҠҠдҪ зҡ„ C д»Јз Ғе’Ң Windows еәһеӨ§зҡ„ DLL зі»з»ҹеә“пјҲKernel32, User32 зӯүпјүжӢјжҺҘеңЁдёҖиө·пјҢз”ҹжҲҗдёҖдёӘеҸҢеҮ»е°ұиғҪиҝҗиЎҢзҡ„В .exeВ ж–Үд»¶гҖӮ
